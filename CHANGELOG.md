## [1.0.0] - 2025-02-17

### Added
- Initial release of DD Corpus Analysis Tools
- Core corpus analyzer with comprehensive statistics
- Advanced analyzer with completeness checking
- Consolidated Excel export optimized for pivot tables
- Library holdings reporter with detailed text reports
- Comprehensive documentation and guides
- Command-line interface with 5 commands:
  - `dd-analyze` - Basic analysis
  - `dd-analyze-advanced` - Advanced analysis with multiple CSVs
  - `dd-excel-export` - Excel-optimized export
  - `dd-reports` - Text reports
  - `dd-validate` - Filename validation
- Python API for programmatic access
- Support for 16 newspaper codes
- Automatic publication frequency detection
- Missing issues estimation
- Multi-format exports (JSON, CSV, Excel, TXT)
- Complete test suite
- Comprehensive documentation:
  - User Guide
  - Excel Pivot Tables Guide
  - API Reference
  - Quick Reference
  - Contributing Guidelines

### Features
- Parses standardized newspaper filenames
- Tracks library holdings and provenance
- Identifies missing and duplicate pages
- Estimates missing issues based on publication frequency
- Generates Excel workbooks with 6 pre-formatted sheets
- Exports single consolidated CSV for pivot table analysis
- Provides 23 comprehensive columns for flexible analysis
- Supports analysis by newspaper, year, library, region, and more
